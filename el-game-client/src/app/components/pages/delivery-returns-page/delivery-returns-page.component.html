Frais d'expédition

Petits objets (livrés par une seule personne)
- Livraison le lendemain (commande avant 20 heures) : gratuit
- Livraison le jour même : gratuit
- Temps spécifique (commande avant 18 heures) : 24,99 euros.

Articles volumineux (livrés par 2 personnes ou plus)
- Livraison le lendemain (commande avant 20 heures) : gratuit
- Livraison le jour même : gratuit
- Temps spécifique (commande avant 18 heures) : 29,99 euros.

Livraison gratuite :
Tous les articles peuvent être livrés gratuitement, uniquement en France, Royaume-Uni, Belgique, Italie, Suisse, Pays-Bas, Allemagne, Royaume-Uniet Espagne (du lundi au samedi uniquement pour les articles volumineux).

Créneau horaire spécifique :
Vous pouvez choisir de payer un supplément pour choisir un créneau horaire spécifique pour votre livraison. 24,99 euros pour les petits articles et 19,99 euros pour les grands articles.

Livraison le jour même :
Vous pouvez également choisir de payer un supplément (24,99 €) pour la livraison le jour même d'articles plus petits qui peuvent être livrés par une seule personne, du lundi au vendredi, uniquement dans certains codes postaux et à condition de commander avant midi.

Frais d'installation pour les articles de grande taille :
Si vous souhaitez payer pour l'installation de votre produit, vous pouvez payer un supplément et les différentes options d'installation vous seront proposées lors du passage à la caisse. Les frais d'installation seront inclus dans les frais d'expédition.
